Here users can be generated into the system with CSV files.


The format for every line in a CSV file must conform to one of the following formats, with the first word in the line being either "Doctor", "Nurse", "Admin", or "Patient". The rest of the items are the user's actual information.

Doctor, First Name, Last Name, Email, Password, Hospital

Nurse, First Name, Last Name, Email, Password, Hospital

Admin, First Name, Last Name, Email, Password, Hospital

Patient, First Name, Last Name, Email, Password, Hospital, Insurance Number, Primary Doctor
